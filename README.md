# Bootcamp Santander — Java

Exercícios e desafios do **Bootcamp Santander** na plataforma DIO, com foco em Java e boas práticas de desenvolvimento.

---

## Conteúdo

| Pasta | Tema |
|-------|------|
| `sintaxeJava01` | Sintaxe básica — variáveis, tipos, entrada de dados |
| `EstruturasCondicionais02` | if/else, switch, operador ternário |
| `EstruturasRepeticao03` | for, while, do-while |
| `TratamentoExececoes04` | try/catch, throws, exceções customizadas |
| `SistemaValidacao06` | Projeto de validação de dados com POO |
| `DesafioSintaxe` | Desafio prático de sintaxe |
| `Desafios` | Desafios variados de lógica e algoritmos |

---

## Stack

- Java (JDK 17+)
- Maven

## Como rodar

```bash
# Com Maven
mvn compile
mvn exec:java -Dexec.mainClass="NomeClasse"

# Direto (sem Maven)
javac SintaxeJava.java && java SintaxeJava
```

## Relevância para QA

Boas práticas de código aprendidas aqui — tratamento de exceções, validação de dados, modularização — são diretamente aplicáveis na escrita de testes automatizados robustos com Appium e Selenium.
